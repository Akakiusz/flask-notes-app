# Flask Notes App

A simple notes app built with Python and Flask, with a SQLite database.

This is my first full-stack web application, created for learning purposes.

**Version:** 1.1.0

## Live demo

🔗 [flask-notes-app-6b3w.onrender.com](https://flask-notes-app-6b3w.onrender.com)

> Note: hosted on a free plan, so the app may take ~30 seconds to wake up after inactivity.

## Features

- Add, view, edit, and delete notes (full CRUD)
- Notes stored in a SQLite database
- Form validation (frontend and backend)
- Clean, responsive design with CSS
- Communication between frontend and backend using JSON

## Technologies

- Python
- Flask
- SQLite
- HTML, CSS, JavaScript
- Gunicorn (production server)
- Deployed on Render

## How to run locally

1. Install the requirements:
   pip install -r requirements.txt
2. Run the app:
   python app.py
3. Open in browser:
   http://127.0.0.1:5000
