# Flask Notes App

A simple notes app built with Python and Flask.

This is my first web application, created for learning purposes.

**Version:** 1.0.0

## Features

- Text input field
- Send button
- Displays a greeting response without reloading the page
- Communication between frontend and backend using JSON

## Technologies

- Python
- Flask
- HTML, CSS, JavaScript

## How to run

1. Install the requirements:
   pip install -r requirements.txt
2. Run the app:
   python app.py
3. Open in browser:
   http://127.0.0.1:5000
